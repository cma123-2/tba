⚠ Navegador no compatible
Esta aplicación requiere File System Access API para guardar y cargar grabaciones. Tu navegador no soporta esta funcionalidad.
Los navegadores compatibles incluyen:
  • Chrome / Chromium 87+
  • Edge 87+
  • Opera 73+
Puedes seguir usando la aplicación para grabar y analizar en tiempo real, pero no podrás guardar archivos en el directorio.
